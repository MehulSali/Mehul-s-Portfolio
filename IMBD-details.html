IMDB Dataset(text-based analysis)

Overview
This case study delves into the analysis of Uber ride data to uncover insights into trip patterns, mileage trends, and ride purposes. The dataset comprises 1,156 entries detailing rides taken by users, including trip start and end times, categories, and purposes.

Objective
The primary goal of this analysis was to understand the distribution of trip categories, popular routes, and mileage patterns while addressing missing data effectively.

Steps and Techniques
1 . Environment Setup :
  • Importing libraries such as numpy and pandas and listing files in the Kaggle input directory.
2 . Data Loading :
  • The dataset is loaded from a CSV file located in the Kaggle directory (IMDB Dataset.csv).
3 . Exploratory Data Analysis (EDA) :
  • Viewing the first few rows of the dataset.
  • Displaying individual reviews.
  • Checking the dataset structure and data types using df.info().

Key Findings
Data Deduplication : Identifying and removing duplicate entries to ensure the dataset is clean and concise.
Subsetting : Reducing the dataset to a smaller sample (first 10,000 entries) for quicker analysis.
Text Preprocessing :
  • Converting all text to lowercase for uniformity.
  • Stripping unnecessary whitespaces.
  • Removing HTML tags from reviews using regex.
